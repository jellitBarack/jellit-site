# Deploy gratuito

## GitHub Pages
1. Crie um repositório público chamado `jellit-site`.
2. Faça upload dos arquivos deste projeto.
3. Acesse `Settings > Pages`.
4. Em `Build and deployment`, selecione `GitHub Actions`.
5. Aguarde a execução em `Actions`.
6. Acesse `https://SEU-USUARIO.github.io/jellit-site/`.

## Alternativas gratuitas
- Cloudflare Pages
- Netlify
- Vercel

Para site estático, todas funcionam bem.
